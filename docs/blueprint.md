# **App Name**: INSPIRE X Dashboard

## Core Features:

- Data Fetching: Fetch registration data from Firestore using the provided configuration.
- Participant Count: Display the total count of registered participants.
- Participant List: Display a list of registered participants, showing their roll number and name.
- Error Handling: Implement error handling to gracefully manage potential issues with data retrieval from Firestore.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to evoke a sense of trust and sophistication, aligning with the event's professional context.
- Background color: Light Grey (#EEEEEE) for a clean, modern, and non-distracting backdrop.
- Accent color: Soft Teal (#4CAF50) to highlight key data points and interactive elements, conveying growth and success.
- Body and headline font: 'Inter' (sans-serif) for a modern, clean, and easily readable experience. Its neutrality ensures that the dashboard is objective and accessible.
- Use simple, outlined icons to represent data points and navigation elements, maintaining a clean and efficient UI.
- Implement a responsive layout that adapts to different screen sizes, ensuring a consistent experience across devices.
- Use subtle transitions when loading data or interacting with the dashboard to provide feedback without being distracting.
